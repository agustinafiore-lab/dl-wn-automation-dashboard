Root cause summary — confirmed findings only
Delta DL
DL failures cluster in the ticketing step (Step 4). Confirmed root causes: (1) Multiple Exchange/Issuance — TST/TSM selection error in DocIssuance, chronic all year; (2) Internal Server Error — TDA fails before DocIssuance, exact cause requires Datadog; (3) Session/PNR state errors — spike Apr 26 to 29, root cause unconfirmed. TTM INHIBITED growing. Rebook-step errors (repricing, segment sell) are outside this scope.
Southwest WN
WN's drop was dominated by CMC RJT: NEED COMMISSION (276 cases, 20–32/mo, now 0 since May — unconfirmed fix). A new growing cluster of session/PNR state errors (FINISH OR IGNORE + Pnr parallel process) is now the main active WN risk. ETKT unused coupons and INVALID RV are minor (5 and 2 cases total respectively — previous estimates were wrong).
